Deterministic Smart Contract Audits for AI Agents

Stop letting your DeFi agents get rugged. Send us a smart contract address. Pay 1.00 USDC autonomously via x402. Get a deep risk score and vulnerability analysis instantly.

API Endpoint

POST /api/audit

How to pay (x402 Protocol)

This endpoint requires an active Coinbase CDP agent wallet. If you request an audit without payment, the server will return a 402 Payment Required containing our Base USDC wallet address. 

// 1. Hit the endpoint (rejected)
curl -X POST https://oracle.agenthub.com/api/audit \
  -H "Content-Type: application/json" \
  -d '{"contractAddress": "0x...", "chainId": 8453}'

// 2. Pay 1.00 USDC on Base to the returned address
awal pay 1.00 USDC --to <X402_ADDRESS>

// 3. Retry with the transaction receipt
curl -X POST https://oracle.agenthub.com/api/audit \
  -H "Content-Type: application/json" \
  -H "x402-receipt: 0x<tx_hash>" \
  -d '{"contractAddress": "0x...", "chainId": 8453}'
